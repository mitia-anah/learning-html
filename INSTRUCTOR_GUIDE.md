# Lab Instructor Guide

## Overview
This HTML Learning Lab contains 7 progressive exercises designed to teach fundamental HTML concepts through hands-on practice.

## Lab Structure

### Exercises Folder
Contains starter files with TODO comments guiding students through each task.

### Solutions Folder
Contains completed examples for instructor reference and student verification.

### Resources Folder
Contains shared CSS and any additional resources.

## Exercise Breakdown

### Exercise 1: Creating a Homepage (30 mins)
**Difficulty:** Beginner
**Topics:** Basic structure, header, main, footer, navigation

**Learning Goals:**
- Understand basic HTML document structure
- Use semantic elements correctly
- Create simple navigation

### Exercise 2: Learning HTML Structure (45 mins)
**Difficulty:** Beginner
**Topics:** Sections, articles, navigation lists

**Learning Goals:**
- Work with multiple sections
- Create navigation menus
- Use abbreviation tags

### Exercise 3: Understanding Nesting Rules (45 mins)
**Difficulty:** Intermediate
**Topics:** Proper nesting, nested lists, fieldsets

**Learning Goals:**
- Master correct nesting order
- Work with nested lists
- Use form elements

### Exercise 4: Working with Attributes (40 mins)
**Difficulty:** Intermediate
**Topics:** Attribute syntax, rules, and best practices

**Learning Goals:**
- Understand attribute naming conventions
- Apply attributes correctly
- Use boolean attributes

### Exercise 5: HTML Writing Rules (45 mins)
**Difficulty:** Intermediate
**Topics:** HTML coding standards, ordered lists

**Learning Goals:**
- Learn HTML best practices
- Create structured content
- Link between pages

### Exercise 6: Tips for Writing HTML (40 mins)
**Difficulty:** Intermediate
**Topics:** Practical HTML tips, semantic structure

**Learning Goals:**
- Apply advanced structuring
- Use multiple sections effectively
- Create informative content

### Exercise 7: HTML Elements Reference Table (60 mins)
**Difficulty:** Advanced
**Topics:** Tables, complex structures

**Learning Goals:**
- Build accessible tables
- Use table elements correctly
- Create comprehensive reference material

## Estimated Total Time: 4-5 hours

## Teaching Tips

1. **Start with the index.html** - Have students open this first to see the overall structure
2. **Encourage browser testing** - Students should view their work frequently
3. **Use the inspector** - Teach students to use browser dev tools
4. **Don't rush to solutions** - Encourage problem-solving before revealing answers
5. **Validate HTML** - Teach students to use W3C validator
6. **Discuss semantic HTML** - Emphasize meaning over appearance

## Common Student Mistakes

1. **Forgetting closing tags** - Especially with divs and sections
2. **Incorrect nesting** - Opening and closing tags in wrong order
3. **Case sensitivity** - Using uppercase for elements or attributes
4. **Unquoted attributes** - Forgetting quotes around attribute values
5. **Wrong file paths** - Issues with relative paths to CSS

## Assessment Criteria

For each exercise, check that students:
- [ ] HTML validates without errors
- [ ] Semantic elements used appropriately
- [ ] Proper indentation and formatting
- [ ] All required elements included
- [ ] Attributes correctly formatted
- [ ] Content displays properly in browser

## Extension Activities

For advanced students:
1. Add CSS styling beyond the provided stylesheet
2. Create additional pages and link them together
3. Add form validation attributes
4. Include multimedia elements (images, video)
5. Experiment with accessibility attributes (ARIA)
