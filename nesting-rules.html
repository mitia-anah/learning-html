Description

HTML tags should always be nested in a proper order. How can we reach that goal? If we explain the word nest in HTML, then it means that the tag opened most recently is always the next tag to close.

Bad code

Code that needs to be fixed, I mean if the code has lots of erros, does not work. Code that fails tests, does not work. We cannot use uppercase with element tags. code has bad struction or nesting, won't work. Those are bad code.

Let's see an example of bad code nesting:

<Header>
<H1>Money</H1>
</Header>
As we see in this example that, the element tag should be lowercase but not uppercase. The indenting is not good.

Do this instead:
<header>
  <h1>Money</h1>&
</header>
