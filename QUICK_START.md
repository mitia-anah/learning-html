# Quick Start Guide

## For Students

### Step 1: Get the Lab Files
Download or clone this repository to your computer.

### Step 2: Open index.html
Double-click `index.html` to open it in your browser, or:
```bash
cd learning-html
xdg-open index.html  # Linux
open index.html      # Mac
start index.html     # Windows
```

### Step 3: Choose Your First Exercise
Click on "Exercise 1: Creating a Homepage" from the list.

### Step 4: Open the Exercise File in Your Editor
Navigate to `exercises/01-homepage.html` and open it in your text editor.

### Step 5: Read the TODO Comments
Each TODO comment tells you exactly what to add.

### Step 6: Complete the Exercise
Fill in the HTML as instructed by the TODO comments.

### Step 7: Test Your Work
Save your file and refresh it in your browser to see your changes.

### Step 8: Check the Solution
When done, compare your work with `solutions/01-homepage.html`.

### Step 9: Move to Next Exercise
Repeat steps 3-8 for each exercise!

## For Instructors

### Setup
1. Review the `INSTRUCTOR_GUIDE.md` for detailed teaching notes
2. Familiarize yourself with all exercises and solutions
3. Test all exercise files in a browser

### Delivery
1. Have students clone/download the repository
2. Walk through the structure using `index.html`
3. Guide students through Exercise 1 together
4. Let students work independently on remaining exercises
5. Review solutions as a group

### Assessment
Use the checklist in `INSTRUCTOR_GUIDE.md` to evaluate student work.

## Tips

- **Save often** - Use Ctrl+S (Cmd+S on Mac) frequently
- **Refresh browser** - Press F5 to see your latest changes
- **Use Dev Tools** - Press F12 to inspect your HTML structure
- **Validate your HTML** - Use https://validator.w3.org/
- **Ask for help** - Don't hesitate to check solutions if stuck

## Common Issues

**Problem:** CSS not loading
**Solution:** Check the file path in your `<link>` tag

**Problem:** Links not working
**Solution:** Verify relative paths in `href` attributes

**Problem:** Page looks blank
**Solution:** Make sure all tags are properly closed
