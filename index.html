Assessment Overview

This assessment consists of 7 exercises designed to evaluate your understanding of HTML fundamentals. Each exercise builds on previous concepts and includes TODO comments to guide you.

Assessment Exercises

  1. Exercise 1: Creating a Homepage (30 mins)

    Basic page structure with header, main content, and footer.

  2. Exercise 2: Learning HTML Structure (45 mins)

    Sections, navigation, and structured content.

  3. Exercise 3: Understanding Nesting Rules (45 mins)

    Proper element nesting and nested lists.

  4. Exercise 4: Working with Attributes (40 mins)

    HTML attribute rules and best practices.

  5. Exercise 5: HTML Writing Rules (45 mins)

    Comprehensive list of HTML coding standards.

  6. Exercise 6: Tips for Writing HTML (40 mins)

    Practical tips for HTML development.

  7. Exercise 7: HTML Elements Reference Table (60 mins)

    Create a detailed table of HTML elements.
