HTML Learning Lab

Master the fundamentals of HTML through hands-on exercises

Welcome to the HTML Learning Lab!

This lab is designed to help you learn and practice HTML through a series of structured exercises. Each exercise focuses on specific HTML concepts and techniques.

Lab Exercises

  1. Exercise 1: Creating a Homepage
  2. Exercise 2: Learning HTML Structure
  3. Exercise 3: Understanding Nesting Rules
  4. Exercise 4: Working with Attributes
  5. Exercise 5: HTML Writing Rules
  6. Exercise 6: Tips for Writing HTML
  7. Exercise 7: HTML Elements Reference Table

How to Use This Lab

  1. Start with Exercise 1 and work through them in order
  2. Read the TODO comments in each exercise file
  3. Complete the HTML structure as instructed
  4. Check your work against the solutions folder when finished
  5. Open the HTML files in your browser to see the results

Need Help?

If you get stuck, refer to the solutions/ folder to see completed examples. Try to complete each exercise on your own first before checking the solutions.
