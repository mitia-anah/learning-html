## ğŸ”¢ Exercise Overview

### Exercise 1: Creating a Homepage (30 mins)
Create a basic homepage with header, main content, navigation, and footer.

**Skills:** Basic structure, semantic elements, navigation

### Exercise 2: Learning HTML Structure (45 mins)
Build a page with multiple sections and navigation lists.

**Skills:** Sections, articles, navigation, abbreviations

### Exercise 3: Understanding Nesting Rules (45 mins)
Practice proper element nesting with nested lists and fieldsets.

**Skills:** Correct nesting order, nested lists, form elements

**Skills:** Ordered lists, inline elements, internal linking

### Exercise 6: Tips for Writing HTML (40 mins)
Build an informative page with practical HTML development tips.

**Skills:** Multiple sections, semantic structure, content organization

### Exercise 7: HTML Elements Reference Table (60 mins)
Create a detailed table documenting common HTML elements.

**Skills:** Tables, proper table structure, organization
