# Changelog

All notable changes to the HTML Learning Lab project.

## [2.0.0] - 2025-11-20

### Major Restructure - Converted to Lab Format

#### Added
- **exercises/** folder with 7 progressive exercise files
  - Each exercise includes TODO comments for guidance
  - Starter templates for students to complete
  - Clear learning objectives in comments
  
- **solutions/** folder with completed examples
  - Full working solutions for all 7 exercises
  - Reference implementations for instructors
  
- **resources/** folder for shared assets
  - sections.css stylesheet for consistent styling
  
- **New documentation files**
  - README.md - Comprehensive lab setup guide
  - INSTRUCTOR_GUIDE.md - Teaching notes and assessment criteria
  - QUICK_START.md - Fast-track getting started guide
  - .gitignore - Standard ignore patterns
  
- **index.html** - Central landing page
  - Links to all exercises
  - Lab overview and instructions
  - How-to-use guide

#### Changed
- Transformed standalone HTML files into structured lab exercises
- Reorganized file structure for better learning flow
- Added detailed comments and TODO markers for student guidance
- Updated README with clear setup instructions
- Enhanced CSS for better presentation

#### Maintained
- Original content and learning objectives
- Email contact information
- Core HTML concepts and examples

### Exercise Structure

1. **01-homepage.html** - Basic page structure
2. **02-learning-html.html** - Sections and navigation
3. **03-nesting-rules.html** - Proper element nesting
4. **04-attribute-rules.html** - HTML attribute guidelines
5. **05-list-of-rules.html** - HTML writing standards
6. **06-tips-for-writing.html** - Practical HTML tips
7. **07-html-elements-table.html** - Elements reference
