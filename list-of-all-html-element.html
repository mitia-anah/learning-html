HTML5 elements

Definition of Html element

HTML elements are mark-up languages that are used to create documents on the world wide web incorporating text, graphics, sound, video, and hyperlinks. HTML elements also add semantics and formatting to parts of document (e.g., make text bold, organize it into paragraphs, lists and tables, or embed hyperlinks and images). Each element can have HTML attributes specified. Elements can also have content, including other elements and text.

The aside element is for making sidebars that are placed next to the main content.
