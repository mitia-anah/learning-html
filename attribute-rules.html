What the rules for Attribute?

Description

Attributes are used to define additional characteristics of the element such as width and height of image. However, attributes are need to be specified in start tag (or opening tag) and usually consist of name/value pairs. Attributes values should always be enclosed in quotation marks.

Code example of using attributes

We are going to see the image attributes below:

<img src=img_me.jpg></img> (Don't do this)
Do this instead:
<img src="img_me.jpg">
